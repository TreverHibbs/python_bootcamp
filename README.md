# python_bootcamp
A jupyter notebook for a short python workshop.
## set up instructions
Install jupyter
```bash
conda install jupyter
```
Install rise for slide show using conda
```bash
conda install -c conda-forge rise
```
### or
install rise using pip
```bash
pip install RISE
```
navigate to your local repository and start up jupyter notebook
```bash
cd /home/username/Documents/python_bootcamp
jupyter-notebook
```

## using rise to run slide show

start the slide show by selecting the Enter/Exit RISE slide show button in the tool bar or by pressing ALT+R.

navigate the slide show with the arrow keys and press Ctr+Enter to execute code.
